# Wordle-Py

---

_Wordle but in python_

## Installation

1. Go to the releases page.
2. Install zip folder `wordle.zip`
3. Extract files from the zip folder
4. Run `wordle.exe` (if extensions are not visible then just `wordle` and type `Application`)
